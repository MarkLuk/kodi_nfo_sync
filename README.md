# NFO Sync

Small home-brew Kodi Program Add-on to sync with .NFO files supporting 3 sync options:
* ### Export Library
	Export Kodi library to NFO files (overwriting the NFO files with library content). Useful for syncing library change on one (main) Kodi instanse to others.
* ### Import (Scan New)
	Import new .NFO files to Kodi library. Does not overwrite existing library information.
* ### Import (Force Refresh)
	Import all .NFO files to Kodi library, overwriting existing library information. Useful for managing the Kodi library via external tool, like [Tiny Media Manager](https://www.tinymediamanager.org/).
