# NFO Sync

Utility add-on to sync with .NFO files supporting 3 sync options:
* ### Export Library
	Sync library with NFO files (overwriting the NFO files with library content). Useful for syncing library change on one (main) Kodi instanse to others.
* ### Import (Scan New)
	Sync new .NFO files with Library content. Does not overwrite existing library information.
* ### Import (Force Refresh)
	Sync all .NFO files with Library content, overwriting library. Useful for managing the Kodi library via external tool, like [Tiny Media Manager](https://www.tinymediamanager.org/).
